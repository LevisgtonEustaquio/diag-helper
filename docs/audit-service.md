# Audit Service

## Objetivo
Registrar ações relevantes realizadas pelos usuários no sistema.

## Eventos Auditados
- Login
- Ações administrativas
- Alterações críticas

## Estrutura do Log
- Usuário
- Ação realizada
- Tipo de evento
- Detalhes adicionais
- Data e hora
- IP

## Importância
- Rastreabilidade
- Segurança
- Conformidade acadêmica e organizacional
